At Need - Overview

This Overview window is displayed when either of two actions has taken place:

  • A New At-Need case is entered and saved.  (By default, certain fields are pre-populated with information that was entered in the At-Need First Call window.)

  • You select an existing case from the case list.

In this window, you can get an overview of the details related to this case.  For instructions on navigating this window and working with other functions offered, click here.

Navigating the "At Need" Window

  • To move to any other tab, simply click on the tab.

  • To work with the list on the right side of the screen, click on the title (e.g., "Forms Completed").

  • Under the Actions section, you can perform such functions as Save, Cancel, get access to the online Help,  Add, Change, Delete, etc.

  • To display a list of forms and/or memorials that are completed and can be printed, click Show Print Forms.

  • To manage your checklists, click Show Checklists.

Each tab shown is used to enter more information about this case.  For details on each tab, see below:

  • Pre-Need.  If this case was setup as a "pre-need," all the pre-need information is displayed in this window.  NOTE: If this is a New At-Need case, do you not enter anything in this window.

  • Call Info.  This is the same information that was entered in the At-Need First Call window.  If you need to make any changes, you can do so directly through this tab.  Remember to click Save when finished.

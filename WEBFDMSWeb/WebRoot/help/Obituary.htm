Obituary

Use this tab to enter an obituary for the decedent.  The main text entry area works very similar to any computerized word processing program.  You can format text, move text around on the page, add bullets or numbers, increase the emphasis by adding bold or italics, etc.

Simply begin typing in the text are of the window.  Then, use the toolbar buttons (described below) to manipulate various aspects of the text.  When you have entered and formatted your text, click Save.  The obituary will be shown on the Overview tab.

BUTTON

DESCRIPTION

Cut.  Cuts the highlighted text.  If needed, you can then paste the text in a different location.

Copy.  Copies the highlighted text.   If needed, you can then paste the copied text into a different location.

Paste.  Inserts the cut or copied text from either the current document, another text file, or a Microsoft Word document.

Print.  Opens the Print dialog box through which you can print the current document.

 

Undo/Redo.  Allows you "undo" a change you just made to the text.  If you want to return the text to the previous condition, the "redo" button is used.

Select All.  Selects all the text in the current window.

Remove Format.  Removes all formatting from the selected text.  This is used in the event that the formatting you've applied is not correct, and you simply want to return the text to its original, unformatted condition.

 

Insert/Remove Hyperlink.  Inserts a hyperlink to a web site.  

    • To insert a link, select the text you want to associate with a web site link, click the Insert Link button, then complete the dialog box that opens.

    • To remove a link, click on the link, and click the Remove Link button.

Bold, Italic, Underline, Strikeout.  Formats the text with the select enhancement.  

Superscript/Subscript.  Changes the size and position of the selected text so that it either above (super) or below (sub) the main text line.  For example e=mc2 (superscript) or H20 (subscript).

Paragraph Alignment.  Indicates how the text should be aligned: left, center, right or full justified.

 

Lists.  Creates a list of numbers or bullet points. Highlight the list of items, and click the desired button to show either numbers or bullets.

 

Indent.  Either increases or decreases the indentation of the text.

Style.  Applies a pre-defined style to the selected text.  A style is a combination of formatting elements - such as font, size, color, etc. - that are saved together to form the style.

Font.  Applies the selected font to the selected text.

Size.  Determines the size of the font.  

 

Text/Background Color. Opens the Select Color window from which you select the color to apply to the selected text or the overall background.

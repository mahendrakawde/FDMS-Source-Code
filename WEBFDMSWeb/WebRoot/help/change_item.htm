Change Item

Use this function to change (edit) any existing inventory item. Through this function you can also view the quantity on hand, remove an item, receive new inventory, or make an item inactive.

  1. While viewing the Inventory Manager window, and locate the item you wish to change.

  2. Double-click on the item to launch the Catalog Item window.

  3. Make the desired changes. Refer to Add New Item for details on each field.

  4. You may also perform any of the following actions: View On Hand, Remove, Receive, and Inactive.
     

Note: For each of these functions, a new window opens. To return to the current window, click the Exit button.

View on Hand

Use this function to view the inventory on hand for a specific catalog item.

  1. In the Inventory Manager window, select the item for which you want to view inventory.

  2. When the Catalog Item window displays, click the View on Hand button to display the View on Hand window. The quantity on hand for this item (along with other details) is shown.
     

  3. To view the quantity on hand at other locations, move to the Location list and select the desired location.

Remove

Use this function to remove the currently displayed inventory item from the system.

Receive

Use this function to receive new inventory.

  1. In the Inventory Manager window, select the item for which you want to receive inventory.

  2. When the Catalog Item window displays, click the Receive button. The Receive Item window displays.
     

  3. Enter the applicable information in the corresponding fields. For example, enter the quantity received in the Quantity field.

  4. When all entries are made, click the Save button. The window updates and shows the quantity just added under the Location field.

  5. Enter additional quantity for this item (if applicable.) Otherwise, click the Exit button.

Inactivate

Use this function to make an item inactive.

  1. In the Inventory Manager window, double-click on the item you want to make inactive.

  2. When the Catalog Item window displays, click the Inactivate button. The item becomes inactive. (You can verify this fact by returning to the Inventory Manager window, selecting, "Show All Inactive Items.")

To reactive the item, do the following:

  1. In the Inventory Manager window, double-click on the item you want to make active.

  2. When the Catalog item window displays, click the Reactive button.

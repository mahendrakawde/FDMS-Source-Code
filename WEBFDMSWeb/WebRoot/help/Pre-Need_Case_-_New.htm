Pre-Need Case - New

Use this function to enter details for a new, "pre-need" case.

  • From the main FDMS Network window, click the New Pre-Need tab.  The Pre-Need window displays.

While working in this window, note the following:

  • You may click Save at any time to save the data already entered.

  • The calendar icon can be used to select a date.

Most of the fields in this window are self-explanatory.  However, here are a few things to note:

  • Maturity.  The date in this field is entered upon the decedent's death.  Thus, during the initial pre-need process, this field is left blank.

  • Account.  Enter a unique, internal number for your location.

When all selections/entries are made, click the Save button.  The window refreshes and the case is opened and the Overview  tab is displayed by default.

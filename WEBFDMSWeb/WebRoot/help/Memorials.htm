Memorials

Use this function to print memorials.

  1. While viewing the Case Status window, move to the top of the window, and click the Show Print Forms button.

  2. When the Print dialog box displays, click the Memorial button.  The Memorials window displays.

  3. Under the Select Verse section, select the desired verse.

  4. Under the Select Format section, select the desired format.

  5. Click the Print button.

When working with verses, you can also do the following:

  • To edit an existing verse, highlight the verse, click the Edit button, make the desired changes in the Edit Verse window, and click Save.

  • To add a new verse, click the New button, and in the Edit Verse window, enter a description and the verse text.  Then click Save.

  • To copy a verse and then use that verse as the basis for another verse, select the verse you want to copy, click Copy, and make the desired changes in the Edit Verse window.

  • To remove a verse, highlight it and click the Remove button.

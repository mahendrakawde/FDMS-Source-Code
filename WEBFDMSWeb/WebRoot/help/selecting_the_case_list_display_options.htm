Selecting the Case List Display Options

When logging in as a new user, it is necessary to select how you want your Case List to display according to the following:

  • Case Code

  • Contract Number

  • Date of Death (the default)

  • Service Date

  • Name of Deceased

In addition, you specify the number of cases you want to display per screen.

  • To select a display option, click the radio button next to the desired item (e.g., Case Code, Contract Number, Date of Death, etc.)

  • To select the number of cases to display per page, move to the Display ____ Cases per Screen field, and enter the desired number.

  • When all selections/entries are made, click Apply Options.  

If you wish to change these options in the future, you can do so via the menu bar. Select Cases, Case List. See also Case Sorting.

Welcome to the FDMS Network

Davidson Software's FDMS Network is the one solution you need for running an effective and efficient business.  From the Funeral Director to the receptionist, FDMS Network's easy-to-use interface coupled with straightforward instructions allow your entire organization to be successful.

So, where do you start?

  • Your system has come pre-configured with some basic items.  However, there are other items that will require your attention.  Be sure you have a designated a SYSTEM ADMINISTRATOR who can perform the Administrative Management functions.  The majority of these functions are found through the Setup and Inventory menus.

  • The most common tasks you will perform as a USER include entering data for Pre-Need and At-Need Cases and printing Forms.

Using this Help System

At any time while you were working in the program, you can access this online help system by doing either of the following:

  • Select Help, Help Index from the menu bar.

  • Click the Help button shown near the upper right corner of each window.

Hyperlinks

Throughout the system, you'll see blue, underlined words.  These "hyperlinks" can be clicked and one of the following will happen:

  • The current window will change and a screen capture will appear.  To close the screen capture, simply click on the link a second time.

  • The current will remain open and a secondary, new window will open with information or a web site.   To close this window, either click the "X" in the upper right corner to close this window ~OR~ click outside of the popup window to close it automatically.

  • The current window will close and be replaced with a new online help window.  Use the Back button found at the top of the browser window to return to the previous online help window.

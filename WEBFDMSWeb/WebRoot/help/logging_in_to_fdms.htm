Logging in to the FDMS Network

Prior to logging into the FDMS Network system, you must have an assigned User Name and Password. If you do not, contact your System Administrator. See also User Administration.

To log in to FDMS Network, do the following:

  1. Click on the desktop shortcut icon, which looks like this:


    If you do not see the above icon on your desktop, see Creating a Desktop Icon Shortcut for more details.
    The FDMS Network – Member Login screen displays.

  2. In the Name field, enter your User Name.

  3. In the Password field, enter your Password.

  4. Click the Login button.

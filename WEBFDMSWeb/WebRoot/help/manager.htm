Manager

You can manage your inventory from this screen. Select Manager from the Inventory menu selection on the Main Menu.

  • From the menu, select Inventory, Manager. The Inventory Manager window displays.

The Actions you can perform in this window include the following:

  • Add New Item

  • Change Item

  • Inventory Reports

In addition, you can change the type of information that is displayed in the list.

  • In the first field, click the drop-down list arrow, and select whether you want to show ACTIVE, INACTIVE, or ALL available items.

  • In the second field, click the drop-down list arrow, and select the category of items you want to view.

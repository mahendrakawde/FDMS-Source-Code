At-Need Reports

  1. From the main menu, select Reports, At-Need. The Reports window displays.

  2. Select a Report. Select the type of report you want to generate by clicking on the item as shown in the list.

  3. Select a Location. Click the drop-down list arrow, and select the desired location. Note: Even if you only have one location, you need to click the arrow and select it.

  4. Select the Dates From:/To:. Specify the time period for which you want to gather data. In each filed, enter the date in the format MM/DD/YYYY (e.g., 01/06/2005). You may also select the dates by clicking the Calendar Icon shown next to each field.

  5. When all selections/entries are made, click the Generate Report button.

Accounting Interface

Use this function to create an interface file into which you have exported data from FDMS Network that can then be imported into the following accounting software programs:

  • Peachtree (general.csv file)

  • QuickBooks (*.iif file type)

  • Business Works (journal.csv file)

NOTE:  Cases must have charges posted in order for the information to be exported to an interface file.  

  1. From the main menu, select Accounting, Accounting Interface.  The Interface to External G/L System window displays.

  2. At this point in the process, you have 2 choices:

  • Select an existing interface file that was created previously.

  • Create a new interface file.  NOTE: It is only necessary to create one interface file for each outside accounting software program you are using.  

  • To use an existing interface file, do the following:

  • Right-click on the desired interface file.

  • From the shortcut menu that displays, click Save Target As.  The File Download dialog box quickly displays, followed by the Save As dialog box.

  • In the Save As dialog box, move to the Save In field, click the drop-down list, and select the drive, directory, folder, subfolder, etc. where you want to save this file.

  • Next, move to the Save as type field, click the drop-down list arrow, and select All Files.

  • Finally, go to the File name field, and enter the name for this file based upon the guidelines shown at the top of this section - e.g., a QuickBooks file might be "011105 Restland Funeral Home.iif".

  • When all selections/entries are made, click the Save button.

  1. To create a new interface file, do the following:

  • Click the drop-down list arrow and select the location for which this interface file will apply.

  • Click the Make Interface File button.

  • If an interface file has already been created for that location (and is shown in the list at the top of the window), a confirmation dialog box is presented:

  • Click the OK button.  The Interface to external GL System window refreshes and the new interface is shown.

Edit Speed Data

The Speed Data tables are where your "Lookup" data is stored.  Whenever you are choosing data from a drop-down box or a pop-up list box, that data is coming from a Speed Data table.  

To edit Speed Data, do the following:

  1. From the main menu, select Setup, Edit Speed Data.  The Edit Speed Data window displays.

  2. Locate the table you wish to make changes to, highlight it, and click the Open button.  A new window displays and shows all the items that correspond with the selected table.  To view a list of tables and the specific format to use when entering speed data, click here.

At this point, you can do any of the following:

  • Add a new item

  • Change an existing item

  • Delete an existing item

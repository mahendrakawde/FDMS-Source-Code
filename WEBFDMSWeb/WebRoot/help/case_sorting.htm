Case Sorting

The Actions drop-down list found on the case list page allows you to change the criteria for how cases are displayed. Choices include the following:

  • To select the desired Sort by option, click the drop-down list arrow shown at the end of the field, and then select the applicable option (e.g., Sort by Case Code, Sort by Contract Number, etc.) Your selection takes effect immediately on screen.

NOTE
If you click the More button, the Case List Display Options window is shown and you can change the manner in which cases are displayed upon first logging into the system.

"Jump" Button

If you select Sort by Case Code, Sort by Case Number, or Sort by Name of Deceased, a field is displayed along with the "Jump" button. This allows you to quickly jump to a case by typing in information.

In the Jump field, enter the following based upon the sort action selected:

  • Sorted by Name of Deceased. Type in the Last Name. As you type, the first name that matches the character entered will be shown.

  • Sorted by Case Code. Type in the Case Code.

  • Sorted by Contract Number. Type in the Contract Number being sure to include the whole number and, if applicable, the dash.
